# Groq Chatbot

## Overview
The Groq Chatbot is an advanced conversational AI designed to assist users in various tasks through natural language processing.

## Features
- Natural language understanding
- Contextual conversation capabilities
- Integration with multiple platforms
- Customizable responses

## Installation
Follow these steps to set up the Groq Chatbot on your local machine:

1. Clone the repository:
   ```bash
   git clone https://github.com/anttip1197/Groq-chatbot.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Groq-chatbot
   ```
3. Install the necessary dependencies:
   ```bash
   npm install
   ```

## Usage
To start the chatbot, run the following command:
```bash
npm start
```
